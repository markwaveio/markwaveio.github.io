# MarkWave Site v2

这是 MarkWave 的静态站版本，适合走：

- 本地改 HTML / CSS
- 推送到 GitHub
- Vercel 自动部署

## 目录

- `index.html`：主页
- `services/`：产品详情页
- `pay/`：预约 / 付费占位页
- `brand/`：品牌 token 与 logo
- `images/`：图片资源

## Vercel 建议配置

- Repository：`markwaveio/markwaveio.github.io`
- Root Directory：`site-v2`
- Framework Preset：`Other`
- Build Command：留空
- Output Directory：留空

## 发布链路

1. 修改 `site-v2/` 下文件
2. `git push origin main`
3. Vercel 自动生成新部署
