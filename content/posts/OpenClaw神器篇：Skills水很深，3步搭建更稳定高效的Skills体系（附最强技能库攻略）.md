# 为什么我让龙虾调用技能，它还是经常搞不定？

这个是近期大家反馈最多的问题。
1. **技能本身不稳定或依赖易变**
2. **提示词/参数不清晰**
3. **技能链条过长**
4. **上下文过杂**
5. **质量评估缺失**

优化方式（最有效的 5 个）
1. **给“明确目标 + 输出格式”**
“用 summarize 抽取3个观点，按 标题-要点-原文证据 输出。”

2. **缩短链条**
• 能一步做完就一步做完，少做“连环技能”
• 复杂任务拆成 2–3 个小任务而不是 1 个大任务

3. **强制验收标准**
• “不包含来源链接就视为失败”
• “三段式输出，不满足则重试”

4. **固定模板**
• 例如：每次选题都用“问题-证据-结论”结构
• 模板会显著提升稳定性

5. **为关键任务加“复核”**
• 先生成→再让它自检（例如“检查是否有虚构来源”）
• 这是最稳的输出控制手段
