# 三、自动化对话保存脚本，记忆资产就是数字生命

**如果你经历“调教了 3 小时的模型，越来越顺手，一次重启重回失忆新兵蛋子”**
那这个设置你一定要看看，不单单OpenClaw适用，所有的AI模型都通用。

### 记忆资产就是你的数字生命
## 为什么要记录？ 避免失忆&数据丢失

#### OpenClaw 的记忆存储机制
- 你跟Openclaw的对话是一个个单独的Session，会作为短期记忆存储在上下文（Context），只要没达到上限，就可以一直存储；同一个Session中，切换模型，记忆也会通用
- 新开session/重启；上下文就会丢失，模型就是一个失忆的新兵蛋子，除非授命主动去读取记忆文件
- 硬盘日志Logs：openclaw会自动保存日志，但这个事给程序看的原始数据，很难二次利用/发给模型

也就是说，你跟模型的对话仅仅是临时存储，随时可能会消失！所以我们必须强制OpenClaw保存好这些对话记忆。

# 四、结构化记忆层级，别被龙虾吃光你的Token

当然OpenClaw有能力记下所有的事情，但是你的模型要付出更大的代价消化掉你所有的记录！分分钟刷爆你的Token。

一件非常必要的操作：
#### > 结构化记忆层级：**让模型“优先读取最重要的事，其它的要用的时候再调取**

你可以让你的OpenClaw按照下面的方式构建记忆资产：

---
#### 多层记忆架构
- **【核心记忆】**：在《你指定的路径》创建一个Core_Profile.md 文件，存储最重要的信息及索引，未来重启/切换模型，第一时间读取这个文件，让模型快速明白我是谁？我是做什么的？
- 【日常记忆】：我们的日常对话请自动保存为MEMORY.md，作为归档备份，方便未来检索记忆。当我与模型对话提到一些关键词时，可以启动调取，以便模型更好的理解并输出结果。
- 【被动归档】：当我觉得我们的对话非常有价值，我会让你将对话转化成**Skill/笔记**，未来我有需要请你发给模型读取操作。
---

#### 这套机制的优点：
✅【降低模型Token的消耗】
- 每次模型仅读取核心记忆文件，大概了解你是谁，你做什么？你的索引关键词，以及深入了解你可以调用读取哪些文档。
✅【优化上下文】
- 模型的上下文限制不同，当上下文达到80%时，模型就会变得非常卡顿反应慢，这种分级调取能更好的优化上下文使用量，避免快速撑爆。
✅ 【检索容错机制】
- 如果某天你想调取之前的聊天内容，给模型参考，可以指示模型读取日常记忆关键词，调取之前的对话记录；或者将你的Skill/手动笔记发给模型，让模型输出更符合你要求的结果。

这些文档你都可以手动维护，让OpenClaw读取更新。


# 五、版本控制配置教程

花了这么多精力调试的 clawdbot.json 和 plist，一旦改错一个标点符号导致系统起不来，后果就很严重。

OpenClaw 自带 GitHub 技能，直接调用Skill来做 **【Git 版本控制】**

把你的配置文件夹 ~/.clawdbot 变成一个 Git 仓库。这样你就有了一个 **“无限撤销键”**。

#### 🛠️ 操作步骤 (只需执行一次)
初始化 Git 仓库，并配置好忽略规则（不备份垃圾日志，只备份核心配置）。

1. **初始化仓库**: 在 ~/.clawdbot 建仓。
2. **配置忽略 (.gitignore)**: 忽略 logs/ 和临时文件。
3. **首次提交**: 备份当前的完美状态。

#### 🕹️ 如何使用「一键回滚」 (Git Cheat Sheet)
把这 3 个命令（记在备忘录里），关键时刻能救命。
1. 每次改配置前：**存档 (Save Game)**
比如你要改模型、改代理、改端口前，先执行：
```
cd ~/.clawdbot
git add .
git commit -m "Backup before changing proxy settings"
```
_(把引号里的字改成你记得住的备注)_

2. 改崩了？**回滚！ (Load Game)**
如果改完配置 Clawdbot 挂了，或者行为异常了，**一键还原**：
```
cd ~/.clawdbot
git checkout .
```
_这会把所有文件恢复到你上一次_ commit _的状态。_
**⚠️ 注意**：回滚后，记得重启服务让它生效：
./clawd_control.sh restart (或者 launchctl kickstart ...)

3. 查看修改了什么 **(Check Changes)**
如果你忘了自己改了哪，想看看差异：
```
cd ~/.clawdbot
git diff
```

**有了这个，你可以大胆地去折腾了。** 只要记得“先存档，再折腾”。💊
